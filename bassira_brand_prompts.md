# Bassira — Brand Prompts

---

## Prompt 1 — Logo Bassira

```
Generate a logo for "Bassira" (بصيرة), an AI-powered multi-agent simulation 
platform for strategic forecasting. Arabic name meaning "insight" or "foresight".

BRAND POSITION: B2B SaaS for C-Level decision-makers in MENA + international 
institutions (governments, funds, think tanks). Premium, credible, modern.

PALETTE CAUSSE — Warm Cultural Resonance:
Primary:   #FF8551  (warm orange — sun, vitality, decisiveness)
Secondary: #006D44  (deep mint-green — growth, stability, trust)
Surface:   #FAF7F2  (cream off-white — paper, clarity, premium)
Text:      #241915  (warm charcoal-brown — authority without coldness)

COGNITIVE ADHESION REQUIREMENTS:
- Evoke "clarity from complexity" — the feeling of seeing through fog
- The mark must read as intelligent + trustworthy, not playful
- Use the metaphor of an EYE or a LENS (بصيرة = sight) without being 
  literal/cliché. Abstract preferred.
- Geometric precision: sharp inner form with soft outer container 
  (reflects the brand duality — rigorous analysis, human warmth)
- Works monochrome, inverted, and at 16px favicon size
- No gradients. Flat vector. 2–3 colors max.

TYPOGRAPHIC LOCKUP:
- Wordmark: "BASSIRA" in Outfit SemiBold, tight letter-spacing (-0.03em)
- Optional sub-wordmark: "بصيرة" in Tajawal Light, right-aligned
- The mark should work standalone (icon only) and paired with wordmark

FORMAT: SVG-ready flat vector, white background preview, dark background 
preview, favicon crop (32×32).
```

## Prompt 2 — Brand DNA complète

```
Build the complete Brand DNA for BASSIRA (بصيرة), an AI multi-agent simulation 
platform for strategic forecasting. Output a structured brand guide covering 
every section below. Be exhaustive, specific, and culturally calibrated.

━━━ CONTEXT ━━━
Product: SaaS platform where C-Level executives run multi-agent simulations 
to forecast how decisions will ripple through political, market, and social 
systems before acting. Think "war-room stress-testing for strategy".

Primary markets: Morocco, GCC, France, UK, pan-African institutions.
Stage: Early commercial (first paying clients, not yet Series A).

━━━ PRIORITÉS ÉDITORIALES ━━━

Très élevé — Définir le ton éditorial
Le ton de voix guide la rédaction sur toutes les plateformes et augmente 
la cohérence de marque. C'est le livrable non-négociable de cet exercice.

Très élevé — Détailler les points de douleur
Les pain points de l'audience calibrent les hooks et CTA de chaque post. 
Sans eux, le contenu reste générique et ne convertit pas.

Très élevé — Préciser la culture cible
La matrice Causse adapte les couleurs aux codes culturels de l'audience 
MENA + Europe. Une couleur peut rassurer ou trahir selon le contexte culturel.

Très élevé — Définir l'objectif cognitif
L'objectif cognitif détermine les styles visuels et les émotions ciblées 
dans chaque post et chaque touchpoint produit.

Très élevé — Enrichir les détails d'audience
L'âge, la localisation et les pain points permettent un ciblage émotionnel 
plus fin. Sans eux, la personnalisation reste superficielle.

━━━ 1. AUDIENCE PROFILE ━━━
Define the full audience matrix across three tiers:
  Tier 1 — Economic buyer: CFO, CEO, Strategy Director (age 38–58, 
    decision authority, time-scarce, ROI-driven)
  Tier 2 — Champion: Head of Strategy, Chief of Staff, Senior Analyst 
    (age 30–45, builds the internal case, technically curious)
  Tier 3 — Skeptic: CTO, General Counsel, Risk Officer 
    (gatekeepers, must be reassured on methodology + data sovereignty)

For each tier, define:
- Core pain points (3–5 specific, named frustrations)
- Emotional drivers (what keeps them up at night)
- Cognitive biases to address (confirmation bias, authority bias, etc.)
- Hook formula: the one sentence that stops the scroll for this person

━━━ 2. PALETTE CAUSSE — CULTURAL CALIBRATION ━━━
Apply the Causse color-culture matrix for MENA + European dual audiences:

For each primary color, specify:
- Cultural resonance in Arabic/Islamic context
- Cultural resonance in French/European context  
- Cognitive trigger (which emotion/state it activates)
- Dominant use case (where it appears in the UI/content)
- Forbidden use cases (where it would misfire culturally)

Colors to calibrate:
  #FF8551  Warm Orange (primary action, confidence)
  #006D44  Deep Mint-Green (secondary, trust, growth)
  #FAF7F2  Cream (background, premium feel)
  #241915  Warm Charcoal (text, authority)
  #A13F0F  Deep Terracotta (CTAs, urgency without aggression)

━━━ 3. COGNITIVE OBJECTIVE ━━━
Define the primary cognitive objective for each touchpoint:

  Landing page hero: [objective + target emotion + visual style directive]
  Product demo / simulation view: [objective + target emotion]
  Pricing page (/offres): [objective — justify premium, reduce friction]
  Calibration page (/calibration): [objective — build methodological trust]
  Email outreach (cold + nurture): [objective + tone shift between cold/warm]
  LinkedIn content: [objective + content archetype mix]
  Pitch deck (investor + client): [objective — dual audience, same deck]

━━━ 4. EDITORIAL TONE OF VOICE ━━━
Define the full tone matrix:

Core personality: 3 adjectives that are OWN-ABLE (not generic like "innovative")
Anti-persona: what Bassira must NEVER sound like (3 anti-archetypes)

Tone registers for each channel:
  Website copy: [register, sentence length, vocabulary level, CTA style]
  In-product UI: [register — commands, labels, empty states, error messages]
  LinkedIn (thought leadership): [register, hook structure, POV intensity]
  Cold outreach (email/LinkedIn DM): [register, opener formula, ask style]
  Arabic content: [tone shifts from French/English — formality level, 
    use of fusha vs darija, honorifics for executives]
  Crisis/sensitive communication: [register when something goes wrong]

Voice guardrails (5 dos, 5 don'ts with examples):
  ✓ "We run 10,000 scenarios so you don't have to." 
  ✗ "Our cutting-edge AI platform leverages..."

━━━ 5. PAIN POINTS × HOOKS MATRIX ━━━
For each of the 5 primary pain points below, generate:
  - The raw fear (emotional core)
  - The rational articulation (how they say it in a board meeting)
  - The Bassira hook (how we address it in ≤12 words)
  - The proof point (what in the product resolves it)

Pain points to cover:
  1. "I'm making a €50M decision based on two consultants' slides"
  2. "Our strategy team disagrees — no shared ground truth"
  3. "We were blindsided by a scenario everyone now says was 'obvious'"
  4. "I can't stress-test a merger before it's announced — too sensitive"
  5. "Forecasting tools give probabilities but never tell me WHY or WHO"

━━━ 6. NAMING & MESSAGING ARCHITECTURE ━━━
Product naming system:
  - Master brand: Bassira / بصيرة
  - Feature names: propose a naming convention for simulation types 
    (Crisis Drill, Policy Brief Stress Test, etc.) — extend the pattern 
    to 8 product SKUs following the same logic
  - Tagline options: 3 options in English, 3 in French, 2 in Arabic — 
    each targeting a different cognitive register (rational / emotional / 
    authority)
  - Domain claim: the one sentence that fits in a <meta description>

━━━ 7. VISUAL SYSTEM DIRECTIVES ━━━
Photography & illustration style:
  - Approved visual metaphors (what imagery is ON-brand)
  - Forbidden visual metaphors (what is OFF-brand and WHY)
  - Data visualization aesthetic (chart style, annotation style, density)
  - Brand pattern / texture (geometric motif derived from the logo mark)

Typography hierarchy in content:
  - Outfit: when and how (sizes, weights, contexts)
  - Manrope: when and how
  - Tajawal: Arabic content rules (bidirectional layout principles)
  - JetBrains Mono: code/data contexts only — rules for use

━━━ 8. CONTENT PILLARS (LinkedIn / Thought Leadership) ━━━
Define 4 content pillars with:
  - Pillar name + positioning angle
  - Content archetype (case study / POV / data post / story)
  - Hook formula specific to this pillar
  - Cadence and optimal format
  - 3 concrete post ideas for each pillar

━━━ OUTPUT FORMAT ━━━
Return a structured document with headers, tables where applicable, 
and concrete examples for every rule. No generic advice. 
Every recommendation must be specific to Bassira's audience, market, 
and product category. Flag anything that requires a cultural consultant 
review for the Arabic market.
```
